## PART 1 概述

其实学习一下 Unity Editor 相关的知识还是很有必要的，毕竟如果想要开发一些插件什么的，做一些酷炫的自定义界面。或者只是在原有插件功能上做一些人性化的改进，都有必要对 Unity Editor 下手。这节课就从最简单且直观的 Editor Window 入手，来尝试搓一个简单的界面吧。
